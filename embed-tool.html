Embed JSON for Deployment

1
Go to your Story Mapper board and click 💾 Download JSON to export your current data.
2
Open the downloaded .json file in a text editor, select all, and paste it into the box below.
3
Click Generate Deployed HTML — a new index.html will download with your data baked in. Push that file to GitHub Pages.
✅ Done! Your file story-mapper-deployed.html has been downloaded.
Rename it to index.html, push it to your GitHub Pages repo, and your board will load with the pre-filled content.
⚠️ Note: GitHub Pages serves static files — visitors can still edit the board in their browser, but changes are stored in their localStorage only. To update the deployed content, repeat this process and push a new file.
